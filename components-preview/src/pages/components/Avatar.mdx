# Avatars

This UI library was created to full a niche in my other project [Neutrino-Chat](https://github.com/maeek/).

## Avatar

Component that displays user's avatar. There are 2 types of avatar. You can specify it by adding `type` prop or by importing 
`AvatarCircle` or `AvatarSquare` directly.

<Avatar text="Avatar component" />


### Props

| Prop                   | Type                                | Required  | Default          |
| ---------------------- | ----------------------------------- | --------- | ---------------- |
| **alt**                | `string`                            | no        | --               |
| **children**           | `ReactNode`                         | no        | --               |
| **className**          | `string`                            | no        | --               |
| **draggable**          | `boolean`                           | no        | true             |
| **fallbackToText**     | `boolean`                           | no        | true             |
| **loader**             | `ReactNode`                         | no        | --               |
| **onClick**            | `Function`                          | no        | --               |
| **onError**            | `Function`                          | no        | --               |
| **onImageLoad**        | `Function`                          | no        | --               |
| **options**            | `AvatarProperties`                  | no        |                  |
| **preloadImage**       | `boolean`                           | no        | `true`           |
| **renderImage**        | `boolean`                           | no        | `true`           |
| **src**                | `string` or `Blob` or `ArrayBuffer` | no        | fallback to text |
| **style**              | `any`                               | no        | --               |
| **text**               | `string`                            | yes       | --               |
| **theme**              | `dark` or `light`                   | no        | `dark`           |
| **title**              | `string`                            | no        | --               |
| **type**               | `square` or `circle` or `undefined` | no        | `square`         |

##### alt
Alternative text to display instead of image. (HTML <img /> tag attribute)
***

##### children
React children.
***

##### className
Adds className to wrapper element.
***

##### draggable
Specifies whether the image can be draggable or not.
***

##### fallbackToText
Enables or disables rendering of fallback image onError or in case the `src` is empty.
***

##### loader
React component displayed while loading image. Gets replaced when image loads.
***

##### onImageLoad
Callback to invoke after image loads.
***

##### onError
Callback to invoke when error occurs during image loading.
***

##### onClick
On click event function.
***

##### options
Options for fallback image as well as for image container itself.

- size - Sets dimensions of thecontainer and fallback image
- background - Sets background for container and fallback image
- color - Sets color for container and fallback image
- maxLength - Specify how many first letters to display on fallback image
- preloadGoogleFonts - This option specify if generation of fallback image should occur after font load. This ensures that
all fallback images will look the same
- font - Specify family, size and weight of a font
```typescript
interface AvatarProperties {
  size: {
    width: number;
    height: number;
  };
  background: string;
  color: string;
  maxLength: number;
  preloadGoogleFonts: boolean;
  font: {
    weight: string | number;
    size: string | number;
    family: string;
  };
}
```
###### Default values
```javascript
const options = {
  size: {
    width: 300,
    height: 300
  },
  background: '#2B3036',
  color: '#5EA8ED',
  maxLength: 3,
  preloadGoogleFonts: false,
  font: {
    weight: 'bold',
    size: 'auto',
    family: 'Roboto, sans-serif'
  }
}
``` 
***

##### preloadImage
Fetch image before displaying it in DOM. Displays loader while loading `src`. Otherwise the `src` of the image will be set directly.
***

##### renderImage
Specify if the component should render <img /> tag. Displays loader or children when disabled.
***

##### src
Image resource, this can be a string, Blob or ArrayBuffer
***

##### style
Pass CSS properties to wrapper element
***

##### text
Text for generating fallback image, used when `src` is not present. Can be disabled by setting `fallbackToText` to false.
Only first letters of words will be placed onto image. This prop is __required__.
***

##### theme
Sets which color palette will be used (`dark` or `light`)
***

##### title
Title to display on hover (browser tooltip)
***

##### type
There are 2 types of this component, one is `square` and the second is `circle`. This specify the appereance of the container.
Default value is `square`.
***


The rest of props will be passed to wrapper element

### Disable preloading image

Avatar on the left has image preloading enabled, it loads image in backgound and then attaching `src` to <img /> tag.
Avatar on the right has preloading and fallbackToText disabled.

<Playground>
  <div className="display-container">
    <AvatarCircle
      text='Testing_Avatar'
      src='https://static.suchanecki.me/avatar.png'
      className="avatar"
      preloadImage={true}
    />
    <AvatarCircle
      text='#'
      src='https://this.image.does.not.exist'
      className="avatar"
      preloadImage={false}
      fallbackToText={false}
      theme="light"
    />
  </div>
</Playground>

### Custom image loader
`renderImage` prop can enforce component to show it's loader and children. Aditionally a custom loader can be passed in `loader` prop.
<Playground>
  <div className="display-container">
    <Avatar
      text='Testing_Avatar'
      src='https://static.suchanecki.me/avatar.pngp'
      className="avatar"
      renderImage={false}
      loader={<div className="custom-loader"></div>}
      options={{
        size: {
          width: 100,
          height: 100,
        }
      }}
    />
  </div>
</Playground>

## Badge
Badge extends usage of `Avatar` component by adding an indicator badge which can display status by changing color or number of unread notifications.

### Props


| Prop                   | Type                                | Required  | Default          |
| ---------------------- | ----------------------------------- | --------- | ---------------- |
| **avatarClassName**    | `string`                            | no        | --               |
| **backgroundColor**    | `string`                            | no        | --               |
| **color**              | `string`                            | no        | --               |
| **indicator**          | `string` or `ReactNode`             | no        | --               |
| **indicatorOnClick**   | `string`                            | no        | --               |
| **status**             | `string`                            | no        | `offline`        |

##### avatarClassName
Pass className to underlying `Avatar` component.
***

##### backgroundColor
Enforce indicator backgroundColor.
***

##### color
Enforce indicator color.
***

##### indicator
Pass `string` or `ReactNode` to display as indicator.
***

##### indicatorOnClick
Indicator onClick event.
***

##### status
Select predefined indicator appereance.
